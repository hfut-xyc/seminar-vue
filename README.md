## Introduction
- 报告汇，基于 Vue.js2 开发的 WebAPP
- 界面组件库使用的是有赞提供的 [Vant](https://vant-contrib.gitee.io/vant/#/zh-CN/)
- 服务端的代码移步 [api-seminar](https://github.com/hfut-xyc/api-seminar)

## Run
- 先安装依赖
```
npm install
```
- 打开`vue.config.js`，设置 publicPath 为 '/'
```
npm run serve
```

### Depolyment
- 打开`vue.config.js`，设置 publicPath 为 './'
```
npm run build
```
